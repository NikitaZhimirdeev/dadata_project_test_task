Dadata program

Для пользователей.

Для начала работы с программой необходимо клонировать полностью данный репозиторий
и все файлы должны храниться в одной директории.

Единственное что понадобится для использования программой, установить Python 3.7+. Для
того чтобы скачать необходим перейти по ссылке https://www.python.org/, Downloads > Download the latest version for Windows и
нажать Download Python. Затем просто установить.

После клонирования репозитория себе на устройство и установки Python, запускаем файл
main.py и следуем инструкциям из программы.

Если вы запускаете программу впервые, то необходимо пройти регистрацию, для этого вводим
цифру 1 и жмем Enter. Далее необходимо ввести логин и пароль (для подтверждения действий необходимо каждый раз нажимать Enter)
Следующий шаг, это ввести API_key и secret, данный ключи можно найти после регистрации 
в личном кабинете https://dadata.ru/profile/#info, скопировав ключ вставляем в консоли программы сочетанием клавиш Ctrl+V.
Затем у вас запросят язык на котом хотите получать результат поиска, на выбор дано два варианта
ru - русский язык, en - английский язык.

После регистрации вы сразу попадете в личный кабинет и у вас запросит адрес для поиска координат.
После ввода адреса, для подтверждения адреса нажмите Enter.

Затем вам будут предоставлены варианты возможных адресов, выберите номер того, который максимально вам подходит, 
обратите внимание нажав 0, вы имеете возможность изменить вводимый адрес.

После ввода наиболее подходящего варианта вы увидите координаты данного адреса, так же вам будет предоставлен
выбор для выхода из программы и повторного проверки другого адреса.



======================================================

Для разработчиков.

Данная программа написана на основе стандартных библиотек с использованием одной дополнительной (dadata - официальная библиотека dadata.ru/api)
устанавливается автоматически.
В качестве базы данных используется SQLite3, хранит в себе данные о зарегистрированных пользователях (логин, пароль, API и secret ключи, язык).

main.py - исполняемый модуль.

HEAD_func.py - Содержит в себе функции текстовых сообщений и функцию сценария авторизации-входа

modules.py - Хранит функции исполнения действий и поиска информации.

BD_modul.py - Модуль для работы с БД.
